üîç Decision Tree - Iris Image Predicition Data Wraggling, Model Optimization, Supervised Learning - Image Classification pandas, PIL, seaborn, torch (PyTorch), scikit-learn, scipy, numpy, matplotlib This project focuses on classifying iris flower images using machine learning techniques, starting with data preprocessing steps like normalization using the ResNet18 model for feature extraction and oversampling to address class imbalances. A Decision Tree Classifier is then trained on this preprocessed data for classifying iris flower images into distinct categories. The model's performance is evaluated using K-Fold cross-validation, ensuring its accuracy and effectiveness across training and test datasets. To enhance classification performance, hyperparameter tuning is employed to optimize the Decision Tree Classifier. Finally, the model's efficacy is tested on new iris flower images, with results presented through classification reports, confusion matrices, and visualizations of the decision tree.
üìà Air Quality Classification Jakarta 2023 Data Wraggling, EDA, Supervised Learning - Classification pandas, matplotlib, seaborn, plotly, scikit-learn, scipy Developed several Classification Algorithm Models to classify Air Quality in Jakarta 2023. The classification algorithms used are Decision Tree, K-Nearest Neighbors (KNN), Random Forest, Support Vector Machine (SVM), Naive Bayes, and Neural Network. Then compare the performance and accuracy of each model, to take the best classification algorithm model in classifying air quality based on ISPU Dataset.
